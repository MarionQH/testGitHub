Liste des tags liés à la typographie

  • <strong> : Utilisé pour indiquer du texte important (en gras).
  • <em> : Utilisé pour indiquer du texte mis en emphase (en italique).
  • <u> : Utilisé pour souligner du texte.
  • <mark> : Utilisé pour marquer du texte, souvent pour le surligner.
  • <b> : Utilisé pour mettre du texte en gras sans indiquer qu'il est important.
  • <i> : Utilisé pour mettre du texte en italique sans signification d'emphase.
  • <s> : Utilisé pour indiquer que du texte a été barré.
  • <q> : Utilisé pour inclure des citations en ligne (avec des guillemets).
  • <cite> : Utilisé pour référencer une source ou une citation.
  • <del> : Utilisé pour indiquer du texte supprimé.
  • <ins> : Utilisé pour indiquer du texte inséré.
  • <small> : Utilisé pour indiquer du texte de taille réduite.
  • <sub> : Utilisé pour indiquer du texte en indice.
